PRODUCTION INSTRUCTIONS
-----------------------

   Install Barlow Condensed (700/800) and Inter (500) from Google Fonts.
2. Select all text → Type > Create Outlines (Illustrator) or
   Right-click > Flatten (Figma).
3. Save as SVG with presentation attributes, no font references.
4. Export PNG at 1x, 2x, 3x from the outlined SVG.
